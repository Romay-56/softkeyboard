softkeyboard是一个支持中文输入的软键盘程序，该项目将整个软键盘界面封装到了一个类中，使用时直接创建对象即可，输入的内容，以信号的形式向外发送。(项目配有中文拼音字典文件ChinesePinyin）

该软键盘其特色便是支持中文输入，以及常用词组输入。
中文输入的原理：
初始化时读取拼音字典文件，将对应的汉语拼音与汉字以键值对形式插入到哈希表中，
哈希表支持一键多值，可以实现同音不同字的特点，此外哈希表有着非常高的读取效率，以实现输入时的快速匹配，而不会卡顿。
另外此键盘还可以实现首字母匹配词组，例：爱情ai'qing输入aq，aiq，aiqing均可匹配出爱情。原理同样是因为哈希表中存在对应的键值对，但该键值对并没有在字典文件中，而是在读字典文件时对词组做了处理，当然由于拼音字典键值对较多，接近两万行。所以加上对词组的特别处理相对比较费时，在主频500MHz的arm开发板上完成一次哈希表的完全插入需要大约1.5s，勉强可以接受。

关于键盘的按键包含所有的数字及大小写字母，以及常用的中英文字符。目前提供了三套皮肤，仅仅做了颜色上的变换。中文输入时会单独弹出一个显示区域，包括拼音以及对应候选词。总之键盘的布局样式还能看，功能上基本能应付普通的输入，所有到此暂时结项，以后有时间再升级优化。
